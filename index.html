Projects

Natural Language to SQL Query converter

SCII, Japan

A system that takes any schema as its input and builds a model that can understand the natural query, convert it to an SQL query and execute it. The framework encompasses an NLP algorithm to build an SQL query, ML techniques to enrich the natural language query and Elastic Search to identify partial values

Mood Assist, a system for student sentiment analysis by monitoring social media activities

Singapore India Hackathon

An application was developed to help schools and parents to identify the psychological and emotional state of the students by analyzing their social media activities. The application uses ML techniques to monitor and identify any signs of stress, depression or academic/peer pressure in the students in real time, in which case triggers are sent to the schools, parents and local helplines to take preventive measures. The application also analyzes the students' potential based on historic and personal data for signs of drastic deviations. The application has a chatbot that empathizes with the students and allows them to confide with it. The chatbot analyses the situation that triggered the problem in the student and the extent of it.

Fetal Distress recognition using Deep Learning based on time series (FHR and UC)

Smart India Hackathon

An ML algorithm that can classify of the birth of a fetus as distress or normal based on the time series of Fetal Heart Rate (FHR) and Uterine Contractions (UC) Based on these given two time series, several other vital time series and features were extracted and engineered. An Long Short-Term Memory (LSTM) model was used to make a binary classification.
